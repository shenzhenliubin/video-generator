## Executive Summary

这是一个**模块化的内容生产流水线系统**，支持从 YouTube 字幕到视频的端到端自动化处理。

**设计原则:**
- **模块化:** 每个处理阶段独立，可单独优化和替换
- **可扩展性:** 从个人工具到平台产品的平滑演进路径
- **提供商抽象:** 支持 LLM/Image/TTS 提供商的灵活切换
- **风格化:** 风格模板系统，支持保存和复用配置

### 抽象接口

```python
# src/api/base.py
from abc import ABC, abstractmethod

class LLMProvider(ABC):
    @abstractmethod
    async def generate_text(self, prompt: str, **kwargs) -> str: pass

    @abstractmethod
    async def analyze_content(self, text: str) -> dict: pass

class ImageProvider(ABC):
    @abstractmethod
    async def generate_image(self, prompt: str, **kwargs) -> bytes: pass

    @abstractmethod
    def get_supported_styles(self) -> list[str]: pass

class TTSProvider(ABC):
    @abstractmethod
    async def synthesize(self, text: str, **kwargs) -> bytes: pass

    @abstractmethod
    def get_supported_voices(self) -> list[dict]: pass
```

### 流水线编排

```python
# src/core/pipeline.py
class Pipeline:
    def __init__(self, settings: Settings):
        self.settings = settings

        # 创建提供商实例
        self.llm = ProviderFactory.create_llm(...)
        self.image = ProviderFactory.create_image(...)
        self.tts = ProviderFactory.create_tts(...)

        # 创建降级提供商
        if settings.fallback_llm:
            self.fallback_llm = ProviderFactory.create_llm(...)

    async def execute(self, input_data: VideoMetadata):
        results = []
        current_data = input_data

        for stage in self.stages:
            try:
                current_data = await self._run_stage(stage, current_data)
                await self._save_checkpoint(stage, current_data)
                results.append(PipelineResult(
                    stage=stage.__name__,
                    success=True,
                    data=current_data
                ))
            except Exception as e:
                recovered = await self._try_recovery(stage, e)
                if not recovered:
                    results.append(PipelineResult(
                        stage=stage.__name__,
                        success=False,
                        error=str(e)
                    ))
                    break

        return results

    async def _run_stage(self, stage, data):
        if asyncio.iscoroutinefunction(stage):
            return await stage(data)
        else:
            return stage(data)

    async def _save_checkpoint(self, stage_name, data):
        """保存检查点"""
        ...

    async def _try_recovery(self, stage, error):
        """尝试恢复"""
        ...
```
